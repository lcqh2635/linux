在 CSS 过渡和动画中，缓动函数（easing functions）决定了属性值变化的速率，对用户体验影响很大。除了 `cubic-bezier(0.4, 0, 0.2, 1)`（这是 Material Design 的标准缓动曲线），还有许多优秀的缓动函数可供选择。以下是几种常用的缓动函数及其视觉效果：

---

### 一、标准缓动函数（CSS 内置）
| 函数名                | 效果描述                                                                 | 适用场景                          |
|-----------------------|--------------------------------------------------------------------------|-----------------------------------|
| `ease` (默认)         | 快速开始，中间减速，缓慢结束                                             | 通用场景                          |
| `ease-in`             | 缓慢开始，加速结束                                                       | 元素离开屏幕                      |
| `ease-out`            | 快速开始，减速结束                                                       | 元素进入屏幕                      |
| `ease-in-out`         | 缓慢开始和结束，中间加速                                                 | 循环动画                          |
| `linear`              | 恒定速度                                                                 | 机械运动（如进度条）              |

---

### 二、高级贝塞尔曲线（推荐用于主题切换）
以下曲线数值来自业界最佳实践：

1. **Material Design 标准曲线**  
   `cubic-bezier(0.4, 0, 0.2, 1)`
    - 特点：快速响应但平滑结束
    - 效果：元素会快速响应操作（10%时间完成40%变化），然后优雅减速
    - 适合：按钮点击、主题切换等需要即时反馈的场景

2. **平滑入场曲线**  
   `cubic-bezier(0, 0, 0.2, 1)`
    - 特点：完全平滑的进入效果
    - 效果：元素从静止状态自然加速，无突然跳跃感
    - 适合：模态框弹出、页面元素入场

3. **平滑离场曲线**  
   `cubic-bezier(0.4, 0, 1, 1)`
    - 特点：保持动量直到消失
    - 效果：元素离开时保持速度，最后快速消失
    - 适合：通知消息消失、元素移除

4. **自然弹跳曲线**  
   `cubic-bezier(0.34, 1.56, 0.64, 1)`
    - 特点：轻微超调后回弹
    - 效果：类似弹簧的轻微弹性效果（但不夸张）
    - 适合：活泼的交互元素（如点赞按钮）

### 四、在 Tailwind 中配置自定义缓动
```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      transitionTimingFunction: {
        'material': 'cubic-bezier(0.4, 0, 0.2, 1)',
        'smooth-in': 'cubic-bezier(0, 0, 0.2, 1)',
        'elastic': 'cubic-bezier(0.34, 1.56, 0.64, 1)'
      }
    }
  }
}
```
使用方式：
```html
<div class="transition-all duration-300 ease-material">
  <!-- 使用Material Design曲线 -->
</div>
```

### 五、针对主题切换的推荐方案
```css
/* 最佳实践：快速响应+平滑结束 */
body, body * {
  transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
  /* 或使用更柔和的版本 */
  /* transition-timing-function: cubic-bezier(0.25, 0.1, 0.25, 1); */
}
```

不同场景建议：
- **暗色模式切换**：`cubic-bezier(0.4, 0, 0.2, 1)`（平衡响应速度与舒适度）
- **元素入场**：`cubic-bezier(0, 0, 0.2, 1)`（避免突兀出现）
- **元素悬停**：`cubic-bezier(0.25, 0.1, 0.25, 1)`（轻微滞后感更自然）

---

### 六、性能注意事项
1. 避免使用 `all` 过渡（如 `transition: all 0.3s`），明确指定属性
2. 缓动函数对 GPU 加速属性（如 `opacity`, `transform`）影响最小
3. 复杂曲线可能在低端设备上导致帧率下降

通过选择合适的缓动函数，可以让您的主题切换既流畅又富有质感！
