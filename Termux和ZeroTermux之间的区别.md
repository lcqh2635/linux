**ZeroTermux** 是 **Termux** 的一个**修改版（魔改版）**，主要针对中国用户优化，解决 Termux 在**网络访问、软件安装、依赖兼容性**等方面的常见问题。它基于 Termux 开发，但增加了一些额外的功能和优化。  

---

## **🔍 ZeroTermux 和 Termux 的区别**
| 特性         | **Termux（官方原版）** | **ZeroTermux（国内修改版）**           |
| ------------ | ---------------------- | -------------------------------------- |
| **开发者**   | Termux 官方团队        | 国内开发者（非官方）                   |
| **软件源**   | 默认使用官方源（国外） | 替换为国内镜像（如清华源、中科大源）   |
| **网络访问** | 可能受网络限制         | 优化网络访问，减少连接问题             |
| **中文支持** | 需手动配置             | 默认支持更好（部分汉化）               |
| **附加功能** | 仅基础功能             | 内置 `tmux`、`zsh`、`oh-my-zsh` 等工具 |
| **安装方式** | F-Droid/GitHub         | 提供 APK 直装（国内更方便）            |
| **更新频率** | 官方维护，更新较慢     | 国内开发者更新，可能更频繁             |
| **安全性**   | 开源，较透明           | 需信任第三方开发者                     |

---

## **🔹 ZeroTermux 的主要特点**
### **1. 国内优化**
- **软件源替换**：默认使用 **清华源、中科大源**，解决 `pkg install` 下载慢或失败的问题。
- **网络代理支持**：内置代理配置，方便科学上网（适合国内环境）。

### **2. 预装增强工具**
- **Zsh + Oh-My-Zsh**：更强大的 Shell，支持主题和插件。
- **Tmux**：终端多窗口管理，适合服务器运维。
- **Vim/Neovim 增强**：预装插件，开箱即用。

### **3. 中文友好**
- 部分界面汉化，减少英文障碍。
- 文档和社区支持更偏向中文用户。

### **4. 一键脚本**
- 提供**快速安装开发环境**的脚本（如 Python、Node.js、Java 等）。

## **🔹 如何安装 ZeroTermux？**
1. **下载 APK**  
   - GitHub Releases: [https://github.com/hanxinhao000/ZeroTermux](https://github.com/hanxinhao000/ZeroTermux)  
   - 国内网盘（如蓝奏云、123云盘等）。  

2. **安装并运行**  
   - 允许安装“未知来源应用”（Android 设置 → 安全 → 允许未知来源）。  
   - 打开 ZeroTermux，它会自动初始化环境（比 Termux 更快）。  

3. **基本使用**  
   - 命令和 Termux 几乎相同（`pkg install`、`apt` 等）。  
   - 额外功能（如 `zsh`、`tmux`）可直接使用。

## **🔹 适用场景**
### **✅ 推荐使用 ZeroTermux 的情况**
- **国内用户**，Termux 安装软件太慢或失败。  
- **不想折腾配置**，希望开箱即用（如直接支持 `zsh`、`tmux`）。  
- **需要中文支持**，减少语言障碍。  

### **❌ 仍建议使用官方 Termux 的情况**
- **注重安全性**（官方版本更透明，ZeroTermux 需信任第三方）。  
- **需要最新功能**（ZeroTermux 可能滞后于官方更新）。  
- **国际用户**（ZeroTermux 主要优化国内环境）。  

---

## **🔹 总结**
- **ZeroTermux = Termux 国内优化版**，适合**中文用户**和**网络受限环境**。  
- **优势**：软件源加速、预装增强工具、中文支持更好。  
- **缺点**：非官方维护，安全性需自行评估。  

如果你在国内，并且 Termux 使用体验不佳（如下载慢、依赖问题），可以尝试 ZeroTermux！🚀
